# Medicine Reminder Web App (MedAlert)

A simple and user-friendly medicine reminder web application designed to help users remember to take their medications on time. The app focuses on clarity, accessibility, and practicality, making it suitable for everyday use and as a foundation for future digital health solutions.


#  Overview

Medication non-adherence is a common challenge, especially for people with busy schedules or long-term treatments. MedAlert (Web App) helps users:

- Set reminders for their medicines
- Keep track of daily medication schedules
- Reduce missed or delayed doses
- Build better health routines


 # Key Features

- Medicine Reminders: Schedule medicines by time and frequency  
- Simple Scheduling:Daily or custom reminders  
- Medicine Management: Add, view, and manage medicines  
- Responsive Design: Works on desktop and mobile browsers  
- Clean UI: Easy to understand and navigate  


 # Known Limitations / Work in Progress

- Notifications may be limited depending on browser permissions  
- Some UI interactions are still being improved  
- Additional validation and error handling are in progress  


# Tech Stack

- Frontend: React (JavaScript)  
- Styling: CSS / Tailwind CSS  
- Build Tool: Vite  
- State Management: React Hooks + zustand  
- Platform: Web Browser

# Getting Started

Prerequisites

- Node.js (v18 or later recommended)
- npm or yarn
Installation

# bash
Clone the repository

git clone https://github.com/Gapera-1/CapstoneProject.git


# Navigate into the project
cd CapstoneProject/medicine-reminder

# Install dependencies
npm install

# Start development server
npm run dev

Open your browser and visit: http://localhost:5173
